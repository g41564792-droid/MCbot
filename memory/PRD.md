# Москитные Сетки - Система заказов

## Оригинальная задача
Telegram бот для заполнения заказов на изготовление москитных сеток с одностраничной формой заказа и интеграцией с Google Sheets.

## Архитектура
- **Backend**: FastAPI + MongoDB
- **Frontend**: React + Shadcn/UI + Tailwind
- **Интеграции**: Telegram Bot API (inline keyboards), Google Sheets API

## User Personas
1. **Заказчик** - создаёт/редактирует заказы через веб-форму или Telegram, отслеживает статус
2. **Администратор** - управляет заказами, меняет статусы, экспортирует данные

## Core Requirements (Реализовано)
- [x] Тип установки: проемная (наружный/внутренний/встраиваемый), дверная, роллетная
- [x] Размеры: 150-3000 мм, количество 1-30
- [x] Рекомендация импоста при размере > 1200 мм
- [x] Цвет: зависит от типа установки, поддержка RAL
- [x] Крепление: Z-образные/металлические/пластиковые, опция прикручивания
- [x] Полотно: стандартное/антипыль/антимошка/антикошка
- [x] Дата готовности: +1 день по умолчанию
- [x] Добавление нескольких позиций в один заказ
- [x] Авторизация пользователей (JWT)
- [x] Админ-панель со статусами и экспортом
- [x] Экспорт в Google Sheets (автоматический)
- [x] Telegram уведомления с inline keyboards
- [x] Редактирование/отмена заказа (только для новых)
- [x] QR-код для Telegram бота
- [x] Push-уведомления при смене статуса

## Telegram Bot Features
- `/start` - Главное меню с inline кнопками
- `/orders` - Список заказов пользователя
- `/help` - Справка

**Inline Keyboards:**
- Главное меню: Новый заказ, Мои заказы, Связаться, Помощь
- Выбор типа: 5 типов установки
- Выбор полотна: 4 типа полотна с ценами
- Навигация: Назад в меню, Назад к типам

**Push-уведомления:**
- При смене статуса на "В работе"
- При готовности заказа
- При выдаче заказа
- При отмене заказа
