# Москитные Сетки - Система заказов

## Оригинальная задача
Telegram бот для заполнения заказов на изготовление москитных сеток с одностраничной формой заказа и интеграцией с Google Sheets.

## Архитектура
- **Backend**: FastAPI + MongoDB
- **Frontend**: React + Shadcn/UI + Tailwind
- **Интеграции**: Telegram Bot API, Google Sheets API

## User Personas
1. **Заказчик** - создаёт заказы через веб-форму, отслеживает статус
2. **Администратор** - управляет заказами, меняет статусы, просматривает статистику

## Core Requirements
- [x] Тип установки: проемная (наружный/внутренний/встраиваемый), дверная, роллетная
- [x] Размеры: 150-3000 мм, количество 1-30
- [x] Рекомендация импоста при размере > 1200 мм
- [x] Цвет: зависит от типа установки, поддержка RAL
- [x] Крепление: Z-образные/металлические/пластиковые, опция прикручивания
- [x] Полотно: стандартное/антипыль/антимошка/антикошка
- [x] Дата готовности: +1 день по умолчанию
- [x] Примечания к заказу и позициям
- [x] Добавление нескольких позиций в один заказ
- [x] Авторизация пользователей (JWT)
- [x] Админ-панель со статусами и экспортом
- [x] Экспорт в Google Sheets (автоматический)
- [x] Telegram уведомления
