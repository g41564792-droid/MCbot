# Москитные Сетки - Система заказов

## Оригинальная задача
Telegram бот для заполнения заказов на изготовление москитных сеток с полным оформлением внутри бота.

## Нумерация заказов
- Формат: **МС-0001**, **МС-0002**, **МС-0003**...
- Автоинкремент через MongoDB counter
- Отображается везде: бот, админка, Google Sheets

## Telegram Bot Flow
1. `/start` → Главное меню
2. "Новый заказ" → Тип установки (5 вариантов)
3. Выбор полотна (4 типа)
4. **Ввод размеров одним сообщением:**
   - Формат: `ширина высота [кол-во]`
   - Пример: `800 1200 2` или `800 1200`
5. Выбор цвета → крепления
6. Импост (если >1200мм)
7. Телефон (пример: +375295012233)
8. Подтверждение → Заказ МС-XXXX создан!

## Команды бота
- `/start` - Главное меню
- `/orders` - Мои заказы
- `/track` - Отследить заказ по номеру
- `/help` - Справка
- `/cancel` - Отменить
