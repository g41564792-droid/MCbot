# Москитные Сетки - Система заказов

## Оригинальная задача
Telegram бот для заполнения заказов на изготовление москитных сеток с полным оформлением внутри бота (без перехода на сайт).

## Архитектура
- **Backend**: FastAPI + MongoDB
- **Frontend**: React + Shadcn/UI (веб-админка)
- **Telegram Bot**: Полный flow заказа через inline keyboards
- **Интеграции**: Telegram Bot API, Google Sheets API

## Telegram Bot Flow (ПОЛНОСТЬЮ ВНУТРИ БОТА)
1. `/start` → Главное меню (inline keyboard)
2. "Новый заказ" → Выбор типа установки (5 вариантов)
3. Выбор полотна (4 варианта с ценами)
4. Ввод ширины (150-3000 мм) → текстовый ввод
5. Ввод высоты (150-3000 мм) → текстовый ввод
6. Ввод количества (1-30) → текстовый ввод
7. Выбор цвета (зависит от типа: 2-4 варианта + RAL)
8. Выбор крепления (3 варианта)
9. Импост (если размер > 1200 мм) → выбор ориентации
10. Ввод телефона → текстовый ввод
11. Сводка заказа + подтверждение
12. "Добавить ещё позицию" или "Подтвердить заказ"

## Команды бота
- `/start` - Главное меню
- `/orders` - Мои заказы
- `/help` - Справка
- `/cancel` - Отменить текущий заказ

## История статусов
Каждый заказ хранит историю изменений статусов:
```json
{
  "status_history": [
    {"status": "new", "changed_at": "2026-02-23T08:00:00Z"},
    {"status": "in_progress", "changed_at": "2026-02-23T10:00:00Z", "changed_by": "admin_id"},
    {"status": "ready", "changed_at": "2026-02-24T14:00:00Z", "changed_by": "admin_id"}
  ]
}
```

## Push-уведомления
При смене статуса пользователь получает уведомление в Telegram:
- 🔧 "В работе" - заказ принят
- ✅ "Готов к выдаче"
- 📦 "Выдан"
- ❌ "Отменён"
